What is the GCD?

The Greatest Common Divisor (GCD) of two or more integers is the largest positive integer that divides all of the integers without leaving a remainder. For example, the GCD of 12 and 18 is 6.

Code Examples

C++


// C++ program to calculate the GCD of two numbers
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

int main() {
    int a = 48, b = 18;
    cout << "GCD of " << a << " and " << b << " is " << gcd(a, b);
    return 0;
}
