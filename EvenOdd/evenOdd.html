What are Even and Odd Numbers?

An even number is an integer that is exactly divisible by 2 (e.g., 4, 0, 2, 6). An odd number is an integer that is not divisible by 2 (e.g., 3, 1, 7). The difference between even and odd numbers is that even numbers have no remainder when divided by 2, while odd numbers have a remainder of 1.

Code Examples

C++


// C++ program to check if a number is even or odd
#include <iostream>
using namespace std;

bool isEven(int n) {
    return n % 2 == 0;
}

int main() {
    int num = 5;
    if (isEven(num))
        cout << num << " is an even number.";
    else
        cout << num << " is an odd number.";
    return 0;
}

Java


// Java program to check if a number is even or odd
public class EvenOddChecker {
    static boolean isEven(int n) {
        return n % 2 == 0;
    }

    public static void main(String[] args) {
        int num = 5;
        if (isEven(num))
            System.out.println(num + " is an even number.");
        else
            System.out.println(num + " is an odd number.");
    }
}
