What is a Factorial?

A factorial, denoted by an exclamation mark (n!), is the product of all positive integers less than or equal to n. For example, the factorial of 5 (5!) is 5 * 4 * 3 * 2 * 1 = 120.

Code Examples

C++


// C++ program to calculate factorial
#include <iostream>
using namespace std;

int factorial(int n) {
    if(n == 0) return 1;
    return n * factorial(n - 1);
}

int main() {
    int num = 5;
    cout << "Factorial of " << num << " is " << factorial(num);
    return 0;
}
