Perfect Numbers

What are Perfect Numbers?

A perfect number is a positive integer that is equal to the sum of its proper positive divisors, excluding the number itself. For example, 6 is a perfect number because its divisors are 1, 2, and 3, and 1 + 2 + 3 = 6.

Code Examples

C++


// C++ program to check if a number is perfect
#include <iostream>
using namespace std;

bool isPerfect(int n) {
    int sum = 1;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            if (i * i != n)
                sum = sum + i + n / i;
            else
                sum = sum + i;
        }
    }
    return (sum == n && n != 1);
}

int main() {
    int num = 6;
    if (isPerfect(num))
        cout << num << " is a perfect number.";
    else
        cout << num << " is not a perfect number.";
    return 0;
}
