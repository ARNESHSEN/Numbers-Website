Sum of Digits

What is Sum of Digits?

The sum of digits is the sum of all the individual digits in a given number. For example, the sum of the digits of 1234 is 1 + 2 + 3 + 4 = 10.

Code Examples

C++


// C++ program to calculate sum of digits
#include <iostream>
using namespace std;

int sumOfDigits(int n) {
    int sum = 0;
    while (n != 0) {
        sum += n % 10;
        n /= 10;
    }
    return sum;
}

int main() {
    int num = 1234;
    cout << "Sum of digits of " << num << " is " << sumOfDigits(num);
    return 0;
}
